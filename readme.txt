== Description ==

# Gestión de Miembros para Delegación

Con este plugin podrás gestionar las comisiones, y los datos de los miembros de tu Delegación.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/miembros` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
